# Overview
- `fabric-test.sh` for testing and installing boilerplates and binaries from the official Hyperledger Fabric repo.
- `linaw-setup.sh` sets up the blockchain bootstrapper, AKA the thesis concept for users.

## LINAW Setup Script
This part will outline the high-level flow of the shell script.

- Hyperledger Fabric binaries
- C compiler
- Python 3 interpreter
- SoftHSM hardware security module
